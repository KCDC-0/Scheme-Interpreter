# Scheme-Interpreter
Implementation of a scheme interpreter in Python

Implemented with reference to chapter 3 from composing programs and materials from cs61a 


## Purpose
Built with the goal of understanding the structure of interpreters, the Read-Eval-Print loop, and the implementation of multiple frames in an environment

Used as a tool to test scheme code from other projects locally


## Features
- Includes parsing and tokenising functions to convert user input into tokens for evaluation
- Includes Frame class to create an edit environment frames
- Includes recursive eval and apply functions to evaluate tokens
- Includes procedures and built-in functions
